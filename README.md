## 🚀 About The Project

Welcome to the repository of my personal portfolio website! This project is a custom-built, single-page application designed to showcase my engineering journey, technical arsenal, flagship projects, and competitive programming milestones. 

The site features a modern, premium **"Deep Dark & Neon Orange"** aesthetic, leveraging glassmorphism and subtle CSS animations—all built entirely from scratch without the use of heavy external UI libraries.

### ✨ Key Features

- **Premium UI/UX:** Frosted glassmorphism effects, dynamic hover states, and smooth scroll animations.
- **Zero Frameworks:** Built strictly with Vanilla HTML5, CSS3, and JavaScript for maximum speed and performance.
- **Interactive Dashboards:** Custom-built bento grids to display LeetCode, GitHub, and HackerRank statistics visually.
- **Responsive Design:** Flawlessly scales from 4K desktop monitors down to mobile screens using CSS Grid/Flexbox.
- **Integrated Contact System:** Fully functional Formspree integration for direct messaging.
